# Introduction_to_AI_FinalProject
人工智能导论期末大作业-模型训练成果（可运行代码）。

### 任务说明：
在机器学习应用中，基于结构化数据进行预测建模是最基础、也是最常见的任务之一。葡萄酒质量预测问题为典型的监督学习回归任务：模型需要根据一组可观测的理化指标（features），预测对应的人工评分结果（label）。本实验基于 Wine Quality 公开数据集，目标是构建并比较多种机器学习模型，评估不同模型在葡萄酒质量预测任务中的表现差异，并分析数据特征对模型性能的影响。
本实验主要关注以下问题：1.数据分布与特征结构是否适合机器学习建模？2.不同模型复杂度对预测性能的影响如何？3.非线性与集成模型是否能有效提升泛化性能？4.数据本身是否对模型性能形成“上限约束”？

### 数据集说明：实验采用 UCI Machine Learning Repository 中的 Wine Quality 数据集。数据包含 1143 条样本，每条样本对应一款葡萄酒的理化检测结果与质量评分。
输入特征（Features）：共 11 个连续型变量，包括酸度、残糖、氯化物、二氧化硫含量、密度、pH、酒精度等理化指标
输出标签（Label）：葡萄酒质量评分，为离散整数值，实际取值范围为 3–8
该数据集特征维度较低、样本量适中，非常适合作为机器学习中的回归建模与模型比较实验案例。

### 代码说明：
1.数据预处理.py是对获得数据进行预处理的代码，进行了标签类别统计、特征分布描述、特征相关性分析等初步的数据预处理工作。
2.机器学习模型(线性&正则回归+KNN).py是线性回归、正则化回归的模型训练与结果输出。
3.XGBoost和随机森林.py是两个集成学习模型的模型训练与结果输出。
4.XGBoost和随机森林解释变量重要性图.py是对集成学习模型结果的补充，输出特征重要性图表。
